Pre-canned Configurations
=========================

Sometimes, figuring things out yourself is just a giant waste of time. Who
really wants to solve issues that have already been solved? This section of
the documentation aims to prevent that.

Make sure that you have a basic understanding of Nginx configuration before
you copy/paste one of these solutions. It's to your benefit to understand why
these work.

Using Recipes
-------------

Some recipes have similar pieces of configuration sections that can be put into
a single configuration file and pulled in using an include. Please, read the
documentation for include if you're interested in saving some repetition.
